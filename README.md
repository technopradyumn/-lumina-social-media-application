# Lumina // Next-Generation Social Experience

**Lumina** is a premium, next-generation Single Page Application (SPA) social media clone built to showcase advanced modern web design skills. It features a frosted glass interface (glassmorphism), responsive layout grids, and interactive reactive state management.

---

## 🌟 Key Features

1. **SPA Router & Tab Manager**: Complete client-side view router (Feed, Explore, Inbox DMs, Creator Analytics, Settings, Profile) without page refreshes.
2. **Component-Based Architecture**: Organization following industry-standard colocated feature directories (grouping JavaScript logic and CSS stylesheets together inside `/src/components/`).
3. **Interactive DMs & Automated Bot Replies**: Inbox messaging simulator featuring active status markers, notifications counts, and delayed AI smart replies.
4. **Stories timeline player**: Complete stories progress timelines overlays that advance automatically.
5. **Creator Analytics Dashboard**: Custom CSS horizontal and vertical charts showing weekly reach, distribution meters, and active profile status logs.
6. **Portal Accent Customizer**: Profile editor forms with accent preset pickers (Purple, Teal, Sunset Rose, Emerald, Gold) that compile and apply HSL variables globally.

---

## 🛠️ Technical Stack

- **Markup & Layout**: Semantic HTML5 & CSS Grid/Flexbox
- **Design Language**: Frosted Glassmorphism, CSS Custom Variables (HSL Themes), Ambient blur backdrops, and keyframe animations
- **Scripting Logic**: Vanilla JavaScript (ES6+), Native Browser Modules (`type="module"`), decoupled state containers
- **Media CDN**: Professional stock photography integrated via Unsplash Image API

---

## 🚀 Running Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/technopradyumn/-lumina-social-media-application.git
   ```
2. Navigate into the directory and launch a static server:
   ```bash
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` in your web browser.
